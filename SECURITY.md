# Security Policy

## Overview

SOLACE ($SOLACE) is a community-driven Jetton deployed on the TON blockchain.
The smart contract was deployed once and is not upgradeable or mintable after deployment.

Total supply is fixed and transparent.

## Token Details

- Name: SOLACE
- Symbol: SOLACE
- Blockchain: TON
- Standard: TON Jetton
- Total Supply: 1,000,000,000 SOLACE

## Security Guarantees

- ✅ One-time mint only (no additional minting possible)
- ✅ No transfer tax, no hidden fees
- ✅ No honeypot or blacklist logic
- ✅ Standard TON Jetton implementation
- ✅ Ownership is renounced after deployment
- ✅ Liquidity provided on STON.fi

## Audits

This token has not undergone a third-party audit.
The contract behavior is simple, transparent, and verifiable on-chain.

## Reporting a Vulnerability

If you discover a security issue or suspicious behavior, please report it privately:

- GitHub Issues (preferred)
- Or contact the project maintainers via official community channels

Please do not disclose vulnerabilities publicly before they are reviewed.
